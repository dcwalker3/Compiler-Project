Input & Output

User Keyboard Input:

input(string output, string dataTypeOfVariable);

Parameters
  • string output: The output to be displayed to the user.
Returns: The User input in the specified data type
Example:
age = input("Enter your age: ");

User Output to Console:

print(string printStatement);

Parameters
  • string printStatement: The output to be displayed to the user.
Returns: None
Example:
print("Hello World!");
Example With Variable Use:
name = "John";
print("Hello, ${name}!");
