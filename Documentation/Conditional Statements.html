Conditional Statements:

IF Statement:

Notes:

Else blocks are optional, but if you use an else block, you must use an if block. Else blocks must also be placed on a separate line from the closing curly bracket of the if block.

Conditional Statement Evaluation is done using the following operators:
Operator Meaning
= Equal To (Does Not assign values when in a conditional statement)
! Not Equal To
> Greater Than
< Less Than

Parameters
  • string condition: The condition to be evaluated.
  • code: The code to be executed if the condition is true.
Returns: None
Example:
if (age > 18) {
print("You are an adult!");
}
Example:
if (age > 18) {
print("You are an adult!");
}
else {
print("You are not an adult!");
}
