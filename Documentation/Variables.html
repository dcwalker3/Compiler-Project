Variables:

Declaring Variables:

Variables are naturally dynamic in nature. This means that you do not need to specify the data type of the variable. The data type of the variable is automatically determined by the value assigned to it.

Example:
age = 18; // age is an integer
name = "John"; // name is a string
isAdult = true; // isAdult is a boolean
height = 1.75; // height is a float

Variable Data Types:

  • int: Integer
  • string: String
  • float: Float
  • bool: Boolean
